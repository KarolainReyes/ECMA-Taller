ECMAScript

ECMAScript es la especificación donde se mencionan todos los detalles de cómo debe funcionar y comportarse Javascript. De esta forma, los diferentes navegadores (Chrome, Firefox, Opera, Edge, Safari...) saben como deben desarrollar los motores de Javascript para que cualquier código o programa funcione exactamente igual, independientemente del navegador que se utilice.

ECMAScript suele venir acompañado de un número que indica la versión o revisión de la que hablamos (algo similar a las versiones de un programa). En cada nueva versión de ECMAScript, se añaden nuevas funcionalidades, manteniendo Javascript vivo y con novedades que lo hacen un lenguaje de programación moderno y cada vez mejor preparado.

Teniendo esto en cuenta, debemos saber que los navegadores web intentan cumplir la especificación ECMAScript al máximo, pero no todos ellos lo consiguen. Por lo tanto, pueden existir ciertas discrepancias. Por ejemplo, pueden existir navegadores que cumplan la especificación ECMAScript al 80% y otros que sólo la cumplan al 60%. Esto significa que pueden haber características que no funcionen en un navegador específico (y en otros sí).

Además, todo esto va cambiando a medida que se van lanzando nuevas versiones de los navegadores web, donde su compatibilidad ECMAScript suele aumentar.

Versiones de ECMAScript

ES1

La primera versión de ECMAScript, conocida como ES1 o ECMAScript 1, fue lanzada en 1997. Esta versión estandarizó JavaScript, un lenguaje de scripting muy popular, y estableció las bases fundamentales para futuras versiones.

detalles adicionales sobre ES1

  • Fecha de lanzamiento: 1997
  • Características clave: Sintaxis básica, tipos de datos, estructuras de control, funciones y objetos.
  • Objetivo: Proporcionar un lenguaje de scripting interoperable para navegadores web.
  • Impacto: Estableció las bases para el desarrollo web y la programación del lado del cliente.
  • Compatibilidad: La mayoría de los navegadores modernos son compatibles con ES1, aunque algunas características pueden no ser compatibles en versiones más antiguas.
  • Legado: ES1 sentó las bases para el desarrollo de versiones posteriores de ECMAScript, como ES2, ES3 y más allá.
  • Documentación: La especificación de ES1 se encuentra en el sitio web de ECMA International.
  • Ejemplo de código: Un simple programa que muestra un mensaje en la consola.
    • 
                console.log("Hola, mundo!");
  • Recursos adicionales: Tutoriales y documentación sobre JavaScript y ECMAScript están disponibles en línea.
  • Comunidad: La comunidad de desarrolladores ha crecido significativamente desde el lanzamiento de ES1, y hay numerosos foros y grupos de discusión en línea.

ES2

La segunda versión de ECMAScript (ECMAScript 2 o ES2) fue publicada en 1998 y, aunque no introdujo cambios funcionales significativos, estandarizó las especificaciones técnicas para alinearlas con el estándar ISO/IEC 16262. La segunda version fue un paso a la estandarización del lenguaje, aunque sin grandes novedades funcionales

Detalles adicionales sobre ES2

  • Fecha de lanzamiento: 1998
  • Características clave: No se introdujeron nuevas características significativas, pero se realizaron correcciones y mejoras menores.
  • Objetivo: Alinear la especificación con el estándar ISO/IEC 16262.
  • Impacto: Mejoró la interoperabilidad entre diferentes implementaciones de ECMAScript.
  • Compatibilidad: La mayoría de los navegadores modernos son compatibles con ES2, aunque no hay características nuevas que destacar.
  • Legado: Sentó las bases para futuras versiones de ECMAScript, como ES3 y más allá.
  • Documentación: La especificación de ES2 se encuentra en el sitio web de ECMA International.
  • Ejemplo de código: No se introdujeron nuevas características, por lo que el código sigue siendo similar al de ES1.
    • 
                console.log("Hola, mundo!");
  • Recursos adicionales: Tutoriales y documentación sobre JavaScript y ECMAScript están disponibles en línea.
  • Comunidad: La comunidad de desarrolladores ha crecido significativamente desde el lanzamiento de ES2, y hay numerosos foros y grupos de discusión en línea.

ES3

La tercera versión de ECMAScript (ECMAScript 3 o ES3) fue lanzada en 1999 y es una de las versiones más influyentes del lenguaje. Introdujo una serie de características y mejoras que sentaron las bases para el desarrollo moderno de JavaScript.

Detalles adicionales sobre ES3

  • Fecha de lanzamiento: 1999
  • Características clave: Introducción de expresiones regulares, manejo de excepciones (try/catch), métodos de cadena y mejoras en la manipulación de objetos.
  • Objetivo: Mejorar la funcionalidad y la interoperabilidad del lenguaje.
  • Impacto: Se convirtió en la base para muchas aplicaciones web y bibliotecas populares.
  • Compatibilidad: La mayoría de los navegadores modernos son compatibles con ES3.
  • Legado: Sentó las bases para futuras versiones de ECMAScript, como ES4 (que nunca se lanzó) y ES5.
  • Documentación: La especificación de ES3 se encuentra en el sitio web de ECMA International.

ES4

La cuarta versión de ECMAScript (ECMAScript 4 o ES4) fue un intento ambicioso de mejorar el lenguaje, pero nunca se lanzó oficialmente debido a desacuerdos en la comunidad sobre su complejidad y alcance. Se consideró demasiado ambiciosa y se abandonó en favor de un enfoque más gradual.

Detalles adicionales sobre ES4

  • Fecha de lanzamiento: Nunca se lanzó oficialmente.
  • Características clave: Se propusieron muchas características nuevas, como clases, tipos de datos estáticos y módulos.
  • Objetivo: Mejorar la funcionalidad y la interoperabilidad del lenguaje.
  • Impacto: Aunque nunca se lanzó, influyó en el desarrollo de versiones posteriores de ECMAScript.
  • Compatibilidad: No hay compatibilidad, ya que nunca se lanzó oficialmente.
  • Legado: Sentó las bases para futuras versiones de ECMAScript, como ES5 y más allá.
  • Documentación: No hay documentación oficial, ya que nunca se lanzó.
  • Ejemplo de código: No hay ejemplos de código oficiales debido a su falta de lanzamiento.
    • 
                // No hay ejemplos de código oficiales
  • Recursos adicionales: Tutoriales y documentación sobre JavaScript y ECMAScript están disponibles en línea.
  • Comunidad: La comunidad de desarrolladores ha crecido significativamente desde el intento de ES4, y hay numerosos foros y grupos de discusión en línea.

ES5

Detalles adicionales sobre ES5

  • Fecha de lanzamiento: 2009
  • Características clave: Introducción del modo estricto (strict mode), JSON, métodos de array como forEach, map, filter, reduce y mejoras en la manipulación de objetos.
  • Objetivo: Mejorar la funcionalidad y la interoperabilidad del lenguaje.
  • Impacto: Se convirtió en la base para muchas aplicaciones web y bibliotecas populares.
  • Compatibilidad: La mayoría de los navegadores modernos son compatibles con ES5.
  • Legado: Sentó las bases para futuras versiones de ECMAScript, como ES6 (ECMAScript 2015) y más allá.
  • Documentación: La especificación de ES5 se encuentra en el sitio web de ECMA International.
  • Ejemplo de código: Uso del modo estricto y métodos de array.
    • 
                "use strict";
                var arr = [1, 2, 3];
                arr.forEach(function(item) {
                    console.log(item);
                });
  • Recursos adicionales: Tutoriales y documentación sobre JavaScript y ECMAScript están disponibles en línea.
  • Comunidad: La comunidad de desarrolladores ha crecido significativamente desde el lanzamiento de ES5, y hay numerosos foros y grupos de discusión en línea.
Características de ES5:

ES5 introdujo características como:

  • Modo Estricto: Permite una detección más rigurosa de errores durante la ejecución del código.
  • JSON: Introducción del formato JSON (JavaScript Object Notation) para el intercambio de datos.
  • Métodos de Array: Se añadieron métodos como forEach, map, filter y reduce para facilitar la manipulación de arrays.
  • Mejoras en Objetos: Se introdujeron métodos como Object.keys y Object.create para mejorar la manipulación de objetos.

Sexta version de ECMAScript

La sexta versión de ECMAScript (ECMAScript 2015 o ES6) fue lanzada en 2015 y es una de las versiones más importantes del lenguaje. Introdujo una serie de características y mejoras que sentaron las bases para el desarrollo moderno de JavaScript.

Características principales de ES6

  • LET Y CONSTlet y const son nuevas formas de declarar variables, ofreciendo una mayor flexibilidad y control sobre el alcance de las variables en comparación con var. const define variables que no pueden ser reasignadas después de la declaración.
